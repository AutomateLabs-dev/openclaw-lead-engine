# OpenClaw Lead Engine — Internal SOP
## Standard Operating Procedure for Lead Handling

---

## Overview

**Goal:** Convert Facebook commenters into qualified leads and paying customers
**Time Sensitivity:** High (first 30 minutes critical)
**Tools Needed:** Facebook, Messenger, CRM (or spreadsheet), Calendar, Payment links

---

## Stage 1: Comment Monitoring

**Frequency:** Check every 30 minutes during active posting
**Platform:** Facebook Business Suite or native app

### Action Steps:
1. See comment "openclaw" (or variation)
2. Click reply
3. Type: "Just sent you a message 👍"
4. Immediately open Messenger
5. Send DM Script 1

**Timer:** Must reply within 30 minutes (ideally 5)

---

## Stage 2: Initial DM

**Timing:** Within 2-3 minutes of their comment
**Message:** 
```
Hey, saw your comment 👍

What part of that caught your attention most?
```

### Action Steps:
1. Send message
2. Wait for reply (do not double-text)
3. If no reply in 24 hours → Mark as "cold" and move on

---

## Stage 3: Classification

**Timing:** After they reply to Message 1
**Message:**
```
Nice — are you thinking of using something like this for yourself, your business, or for clients?
```

### Action Steps:
1. Send classification question
2. Read their response carefully
3. Categorize:
   - Personal → Path A
   - Business → Path B
   - Agency/Clients → Path C
   - Unclear → Ask clarifying question

---

## Stage 4-A: Personal Path (DIY)

**Signals:** "For myself," "personal project," "side hustle," "learning"

### Action Steps:
1. Ask: "Quick question: are you already using any AI tools (ChatGPT, Claude, etc.) or is this all new?"
2. Based on answer:
   - Experienced → Send CourseSprout link directly
   - New → Explain value, then send link
3. Message:
```
I run a small community where I show people how to set up their own AI team — hosted environment, pre-built skills, the whole thing.

$47/month. Self-serve. You get in, set it up, start building.

[Link]

Questions? Just reply.
```

**If they sign up:**
- Mark in CRM: "Active - DIY"
- Add to onboarding email sequence
- Flag for 7-day activation check

**If they don't sign up within 48 hours:**
- Single follow-up: "Still thinking it over? Happy to answer questions."
- Then let them go cold

---

## Stage 4-B: Business Path

**Signals:** "For my business," "my company," "my team," "we need"

### Action Steps:
1. Ask: "What kind of business are you in?"
2. After they answer, present both options:
```
So there's two ways I work with businesses:

1) I set up the AI team inside your business — hosted, managed, tailored to your workflows. Direct support. Starts at $297/month plus setup.

2) You join the community ($47/month), learn the system, build it yourself. Good if you have someone on your team who likes tinkering.

Which direction sounds more like what you need?
```

**If they choose Service (Option 1):**
1. Ask qualifying questions:
   - "What's the main thing you're trying to automate or scale?"
   - "Timeline — looking to get this moving in the next few weeks, or later this year?"
2. Book strategy call (Calendly link)
3. Mark in CRM: "Hot Lead - Business"
4. Send calendar confirmation

**If they choose DIY (Option 2):**
1. Send CourseSprout link
2. Add note: "Might upgrade to service later"
3. Mark in CRM: "Warm Lead - DIY"

---

## Stage 4-C: Agency Path

**Signals:** "For clients," "my agency," "I build sites," "resell"

### Action Steps:
1. Ask: "Are you thinking of using this to deliver work faster, or to offer AI team setups as a service to clients?"

**If for internal delivery:**
- Treat as Business Path (Stage 4-B)

**If for resale/white-label:**
1. Message:
```
That's exactly where this is heading. White-label AI team setups for your clients.

I'm not fully ready to roll that out yet — still testing with a few people.

But if you're serious about it, happy to have a conversation. The model would be per-seat or bulk pricing, you manage the client relationship.

Want to jump on a quick call this week?
```
2. Book call
3. Mark in CRM: "Hot Lead - Agency Partner"

---

## Stage 5: Objection Handling

### "How much does it cost?" (Too early)

**Response:**
```
Depends on which path makes sense.

If you're DIY: $47/month for the community + hosted setup.

If you want me to build and manage it for your business: starts at $297/month plus setup.

Which direction are you thinking?
```

### "Is this a course?"

**Response:**
```
No — it's an actual system I host for you. You get access to the AI team, pre-built skills, all of it.

The community has training videos, but the main thing is the working environment.

Think of it like getting a software subscription with onboarding, not a "watch these videos" course.
```

### "I need to think about it"

**Response:**
```
Totally fair.

Here's the link if you want to check it out: [Link]

Or if you have questions, just reply. I actually answer these messages myself.
```

**Then:** Do not follow up again. Let them come back.

### "Can I see a demo?"

**Response (Personal):**
```
The post you commented on was basically a demo — that's what the system does.

In the community, I have videos showing exactly how to set it up and use it.

$47 for the first month. If it's not what you expected, cancel anytime.
```

**Response (Business):**
```
For business clients, I do a paid strategy call first — $200, applied to setup if we move forward.

On that call, I'll show you exactly how it would work for your specific situation.

Want to book a time?
```

## Stage 6: CRM Tracking

**Tool:** Simple spreadsheet or Notion

**Columns:**
- Date
- Name
- Source (which post)
- Category (Personal/Business/Agency)
- Status (New → Active → Customer → Upgraded)
- Next Action
- Revenue

**Status Definitions:**
- **New:** Just commented, DM sent
- **Active:** Replying, in conversation
- **Cold:** Stopped replying > 48 hours
- **Customer:** Paid ($47 or higher)
- **Upgraded:** Moved to higher tier

---

## Stage 7: Handoff to Team

**When to hand off:**
- Business client books call → To Dave for strategy call
- DIY customer needs tech support → To community/FAQ first
- Custom system build needed → To Dave + Byte

**Handoff format:**
```
Lead: [Name]
Category: [Business/Agency]
Context: [What they need]
Status: [Where in the conversation]
Next Step: [What needs to happen]
Link to conversation: [Screenshot or thread]
```

## Daily Checklist

**Morning (15 min):**
- [ ] Check overnight comments
- [ ] Reply to all "openclaw" comments
- [ ] Send DMs

**Midday (15 min):**
- [ ] Check for DM replies
- [ ] Respond to active conversations
- [ ] Update CRM

**Evening (10 min):**
- [ ] Final comment check
- [ ] Review next day's active leads
- [ ] Plan tomorrow's post (if posting)

---

## Weekly Review (30 min)

**Metrics to track:**
- Comments received
- DMs started
- Conversations had
- $47 sales
- Business calls booked
- Business deals closed
- Revenue

**Conversion rates:**
- Comments → DMs: Target 100%
- DMs → Active convo: Target 40%
- Active → $47: Target 25%
- $47 → Business: Target 10%

**Adjustment:**
- If comments → DMs is low → Post timing/content issue
- If DMs → Active is low → Opening message needs work
- If Active → $47 is low → Offer/price resistance

---

## Automation Opportunities

**Can automate:**
- Initial DM response (Facebook auto-reply)
- Onboarding emails (CourseSprout)
- 7-day activation check (email sequence)
- CRM updates (Zapier)

**Should NOT automate:**
- Classification questions
- Objection handling
- Business sales conversations
- Strategy calls

**Rule:** Automate logistics, not relationships.
