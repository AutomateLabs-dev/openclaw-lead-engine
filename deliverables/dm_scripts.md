# OpenClaw Lead Engine — DM Conversation Scripts
## Classification Flow: Personal vs Business vs Agency

---

## Message 1 (Immediate, after they comment)

**Timing:** Within 2-3 minutes of their comment

**Script:**
```
Hey, saw your comment 👍

What part of that caught your attention most?
```

**Notes:**
- Open-ended question
- No selling, no links, no pitch
- Wait for their reply before next message

---

## Branch A: They reply with something personal/use case

**Examples:** "The audio thing" / "Research" / "Content creation" / "Saving time"

**Message 2:**
```
Nice — are you thinking of using something like this for yourself, your business, or for clients?
```

**Goal:** Classify into Personal / Business / Agency

---

## Path 1: PERSONAL (DIY User)

**Signal:** "For myself" / "My side project" / "Personal use" / "Just curious"

**Message 3:**
```
Got it. Personal projects are actually a great way to learn the system.

Quick question: are you already using any AI tools (ChatGPT, Claude, etc.) or is this all new?
```

**If experienced with AI:**
```
Perfect. You've got the foundation.

I run a small community where I show people how to set up their own AI team — hosted environment, pre-built skills, the whole thing.

$47/month. Self-serve. You get in, set it up, start building.

Want the link?
```

**If new to AI:**
```
Cool — this is actually a great entry point. Easier than trying to duct-tape 5 different tools together.

I have a $47/month community where I host the system for you, walk you through setup, and you get access to all the pre-built skills.

Plus onboarding videos so you're not figuring it out alone.

Want to check it out?
```

---

## Path 2: BUSINESS (Company/Team)

**Signal:** "For my business" / "My company" / "My team" / "We need"

**Message 3:**
```
Business use — that's where this really shines.

What kind of business are you in? (Helps me understand what systems would actually move the needle for you)
```

**After they answer:**
```
Makes sense. 

So there's two ways I work with businesses:

1) I set up the AI team inside your business — hosted, managed, tailored to your workflows. Direct support, no learning curve for you. Starts at $297/month plus setup.

2) You join the community ($47/month), learn the system, build it yourself. Good if you have someone on your team who likes tinkering.

Which direction sounds more like what you need?
```

**If they want service (Option 1):**
```
Smart. Most business owners don't want to become AI engineers.

Let me ask a couple quick questions to see if we're a fit:

- What's the main thing you're trying to automate or scale?
- Timeline-wise, are you looking to get this moving in the next few weeks, or is this more "sometime this year"?
```

**If they say DIY (Option 2):**
```
Got it — you've got someone who can run with it.

The community is $47/month, hosted environment included. You'll get access to all the skills, plus onboarding videos.

Fair warning though: it's self-serve. I don't do hands-on support at that level.

Still want the link?
```

---

## Path 3: AGENCY (Service Provider)

**Signal:** "For clients" / "My agency" / "I build sites" / "I do marketing"

**Message 3:**
```
Agency use — I like it. That's actually the layer I'm building next.

Are you thinking of using this to deliver work faster, or to offer AI team setups as a service to clients?
```

**If for delivery (internal use):**
```
Smart. Speed and quality without hiring.

The $47 community gives you the system to learn and build. Or if you want it set up properly without the learning curve, I can do a business implementation.

What are you currently delivering to clients? (So I know which systems would actually help)
```

**If for resale (offering to clients):**
```
That's exactly where this is heading. White-label AI team setups for your clients.

I'm not fully ready to roll that out yet — still testing with a few people.

But if you're serious about it, happy to have a conversation. The model would be per-seat or bulk pricing, you manage the client relationship.

Want to jump on a quick call this week?
```

## Common Objections & Responses

### "How much does it cost?" (Early in conversation)

**Response:**
```
Depends on which path makes sense.

If you're DIY: $47/month for the community + hosted setup.

If you want me to build and manage it for your business: starts at $297/month plus setup.

Which direction are you thinking?
```

### "Is this a course?"

**Response:**
```
No — it's an actual system I host for you. You get access to the AI team, pre-built skills, all of it.

The community has training videos, but the main thing is the working environment.

Think of it like getting a software subscription with onboarding, not a "watch these videos" course.
```

### "I need to think about it"

**Response:**
```
Totally fair.

Here's the link if you want to check it out: [CourseSprout link]

Or if you have questions, just reply. I actually answer these messages myself.
```

### "Can I see a demo first?"

**Response (Personal/DIY):**
```
The post you commented on was basically a demo — that's what the system does.

In the community, I have videos showing exactly how to set it up and use it.

$47 for the first month. If it's not what you expected, cancel anytime.
```

**Response (Business):**
```
For business clients, I do a paid strategy call first — $200, applied to setup if we move forward.

On that call, I'll show you exactly how it would work for your specific situation.

Want to book a time?
```

## Classification Cheat Sheet

| Response Pattern | Category | Next Step |
|------------------|----------|-----------|
| "For me/myself" | Personal | $47 community |
| "For my business/company" | Business | Qualify → $297+ service |
| "For clients/agency" | Agency | Reseller convo or DIY |
| Vague/unclear | Probe deeper | Ask qualifying question |

---

## Timing Guidelines

- Reply to comments: Within 30 minutes (ideally 5)
- First DM: Immediately after public reply
- Message 2: After they respond (don't double-text)
- Close or link: Only after classification
- No follow-up spam: If they go quiet, let them

---

## Handoff Triggers

**Move to Business Call:**
- They have a real business
- They want done-for-you
- Budget questions indicate $297+ not a barrier

**Stay in DIY Flow:**
- Personal use
- Price-sensitive
- Wants to learn/build themselves

**Flag for Agency Program:**
- Wants to resell
- Has client base
- Asks about white-label
