# CourseSprout Structure
## Membership Content + Upgrade Triggers

---

## Overview

**Platform:** CourseSprout
**Price:** $47/month
**Goal:** Get users activated and upsold to higher tiers
**Structure:** Core training → Skills library → Advanced systems → Upgrade triggers

---

## Module 1: Welcome + Setup (Week 1)

**Objective:** Get them into their environment and running first workflow

### Lessons:
1. **Welcome to OpenClaw** (5 min)
   - What you now have access to
   - How to log in
   - Where to get help

2. **Your AI Team Overview** (10 min)
   - Meet the agents (Astra, Elon, Gary, Warren, Byte)
   - What each one does
   - When to use which

3. **Environment Walkthrough** (15 min)
   - Dashboard tour
   - How to start a session
   - Basic commands
   - Safety/rules

4. **Your First Workflow** (20 min)
   - Simple task: "Research a competitor"
   - Step-by-step walkthrough
   - Screenshot guide
   - Expected output

5. **Troubleshooting Basics** (10 min)
   - Common issues
   - Where to ask for help
   - How to check if tools are working

**Action Item:** Complete first workflow and post result in community

**Upgrade Trigger:** If stuck > 3 days → Email offering onboarding ($297)

**Objective:** Master essential skills

   - Forms & quizzes

4. **Operations**

**Action Item:** Complete 3 projects using different skills

**Upgrade Trigger:** After 2 weeks, if active → Email about "Accelerator" systems

- Completed first workflow

**Engagement (Goal: 40%)**

- Posted in community
- Completed 3+ projects

**Upgrade (Goal: 15%)**
- Purchased onboarding
- OR booked business call
- OR upgraded to business tier

**Retention (Goal: 80% month 2)**
- Still active after 30 days
